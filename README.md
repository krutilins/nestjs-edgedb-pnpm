## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.
[EdgeDB](https://www.edgedb.com/) open-source database designed as a spiritual successor to SQL and the relational paradigm.
## Installation

```bash
$ pnpm install
```

## Running the db

```bash
# database init
edgedb project init
# generate interfaces
pnpm generate:edgeql
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
